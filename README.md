# 新疆商城-思路乐购  vue3 版本

## 开发环境

npm: 6.14.14

node: v14.17.5

vue: 2.2.6 # TODO 改成3.x

## 步骤

1. 安装nvm

2. 用nvm安装node

3. 用npm安装vue, vue-cli, vue-router等等 ( vue-router > 2.0)

4. 安装sass:

```
$ npm install sass
```

5. clone 项目

6. 运行

```
  6.1 $ npm install
  6.2 $ npm run dev
```

  第一次运行成功以后只需要6.2, 6.3, 运行在localhost:8080端口上

7. 打包代码

```
$ npm run build
```

8. Capistrano（自动化部署） 访问地址：

```
$ bundle exec cap deploy
```
